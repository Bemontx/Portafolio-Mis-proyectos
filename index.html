Sobre mí

Data Scientist con experiencia en ingeniería de datos adquirida en el bootcamp intensivo de Soy Henry, donde desarrollé habilidades avanzadas en análisis de datos, machine learning y desarrollo de modelos predictivos. Formado previamente en Oracle Next Education, consolidé una sólida base para abordar conceptos complejos y aplicarlos en proyectos prácticos. Especializado en ingeniería de datos y con conocimientos en metodologías ágiles, estructura de datos y evaluación de modelos. Busco seguir ampliando mis conocimientos en proyectos de la vida real.

Data Engineer, Analytics-Machine-Learning

Realicé un proyecto que combinó análisis exploratorio de datos (EDA) y un proceso de extracción; transformación y carga (ETL) en datos de reseñas de negocios de restaurantes y hoteles de Yelp y Google. Implementé un pipeline en Google Cloud Platform (GCP) para procesar y almacenar los datos en BigQuery. Desarrollé un dashboard interactivo en Tableau para visualizar insights y un sistema de recomendaciones basado en machine learning con scikit-learn. Utilicé herramientas Pandas, NumPy, Matplotlib, Seaborn, Apache Beam y Cloud Dataflow en el proceso.

Google Cloud(Bucket,Dataflow,Functions,BigQuery) ETL(Python,Extraction,transformation and loading) EDA(Exploratory Data Analysis) Dashboard(Tableau) Machine Learning(Modelo de recomendaciones)

EncryptX

Como líder de este proyecto, dirigí la creación de un programa en Java para encriptar y desencriptar texto. Implementé algoritmos como AES, DES y RSA, además de diseñar una interfaz de usuario intuitiva utilizando la biblioteca gráfica Swing. Mi rol implicó coordinar tareas, garantizar la seguridad del sistema y asegurar una experiencia de usuario fluida. Fue gratificante ver cómo esta herramienta contribuyó a la protección de datos en el mundo digital.

Java JCA (Java Cryptography Architecture) Swing (para la interfaz de usuario) AES, DES, RSA (algoritmos de cifrado)

CoinConvert

En el proyecto Conversor de Moneda en Java, lideré el desarrollo de una aplicación que permite a los usuarios realizar conversiones entre diferentes tipos de moneda de manera rápida y precisa. Utilizando Java y aplicando principios de orientación a objetos, creamos una interfaz intuitiva que facilita a los usuarios ingresar el monto en una moneda y obtener su equivalente en otra seleccionada. Me encargué de diseñar y desarrollar la interfaz gráfica amigable utilizando la biblioteca Swing. Además, implementamos un sistema robusto que garantiza la precisión y fiabilidad en todas las conversiones, gracias al uso de tipos de datos precisos y sólidos principios de orientación a objetos.

Java Swing (para la interfaz gráfica de usuario) Orientación a Objetos
